One family, **Figtree**, everywhere. Hierarchy comes from size, weight and
spacing, never from switching typefaces.

## Type scale

Two registers: **editorial** (Display XL through Headline) is fluid via
`clamp()`; **interface** (Title and below) uses fixed sizes for dense product UI.

<TypeScale />

## Typefaces

| Family | Role | Weights |
| --- | --- | --- |
| Figtree | Everything: display, headings, UI, body | 400, 500, 600 |
| JetBrains Mono | Code, tokens, tabular identifiers | 400, 500 |

<Callout tone="info" title="Loading">
  Both families are self-hosted via <code>next/font</code> with
  <code>display: swap</code> and Latin subsetting. Never load fonts from
  third-party CDNs in product code.
</Callout>

## Usage rules

### Hierarchy

A screen should read as a clear outline when you squint. If two adjacent text
blocks compete, demote one.

<DoDont>
  <Do>
    Use one Headline per page, and step down through Title and Body without
    skipping more than one level.
  </Do>
  <Dont>
    Create hierarchy with color or italics, weight and size are the only
    hierarchy tools in this system.
  </Dont>
</DoDont>

### Line length and rhythm

- Large text tracks tight (Display XL −3.5%), small text needs room (Caption +1%), baked into the scale, never overridden per-instance
- Body text: 45–75 characters per line, line height 1.75 for prose, 1.5 for dense UI
- Never justify text. Left-aligned, ragged-right, always

### Numbers and data

Any column of numbers, order IDs, amounts, timestamps, is JetBrains Mono
with tabular figures.

<DoDont>
  <Do>
    Right-align numeric table columns and set them in mono with tabular
    figures.
  </Do>
  <Dont>
    Mix mono identifiers into running prose, only data surfaces get mono.
  </Dont>
</DoDont>

## Engineering reference

The scale is defined as Tailwind theme tokens, so styles stay consistent and
purgeable.

```css
@theme inline {
  --text-display-xl: clamp(3rem, 7.5vw, 5.5rem);
  --text-display-xl--line-height: 1.02;
  --text-display-xl--letter-spacing: -0.035em;

  --text-headline: clamp(1.75rem, 3.2vw, 2.5rem);
  --text-title: 1.375rem;
}
```

```tsx
<h1 className="text-headline text-ink">Page title</h1>
<p className="text-body-lg text-ink-secondary">Lede paragraph…</p>
<span className="font-mono text-[13px]">NB-20581</span>
```
