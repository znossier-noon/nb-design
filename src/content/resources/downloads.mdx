## Fonts

| Family | License | Formats |
| --- | --- | --- |
| NoonTree | Open (OFL, custom Figtree) | woff2 weights 400/500/600 |
| Noon Arabic | Open (OFL, Cairo) | woff2 weights 400/500/600, install as family name "Noon Arabic" |
| Noon Mono | Open (OFL, JetBrains Mono) | woff2 weights 400/500, install as family name "Noon Mono" |

All three families include brand glyphs `dhm` (U+20C3) and `sar` (U+20C1).

Product surfaces load fonts via `next/font`. These downloads are for Figma
and documents.

## Color palettes

The [token palette](/foundations/color) packaged for every tool:

- **Figma**, included as variables in NB Foundations (preferred)
- **Swatch files**, `.ase` (Adobe), `.clr` (macOS), `.gpl` (GIMP/Inkscape)
- **JSON**, the raw token set from `@noon-business/tokens`

## Tokens for engineers

```bash
npm install @noon-business/tokens
```

Ships CSS custom properties, a Tailwind preset and the raw JSON. Setup guide:
[documentation](/resources/documentation).
