## Fonts

| Family | License | Formats |
| --- | --- | --- |
| noontree | Brand font | Pending approved production files |
| Figtree | Web fallback | Loaded until noontree files are added |
| JetBrains Mono | Open (OFL) | Variable TTF, weights 400/500, woff2 |

Figma defines `noontree` as the primary family. Product surfaces should use the
approved self-hosted noontree files once available; the site currently keeps a
web fallback so pages remain stable.

## Color palettes

The [token palette](/foundations/colors) packaged for every tool:

- **Figma**, included as variables in NB Foundations (preferred)
- **Swatch files**, `.ase` (Adobe), `.clr` (macOS), `.gpl` (GIMP/Inkscape)
- **JSON**, the raw token set from `@noon-business/tokens`, light and dark

## Presentation kit

- Slide templates (16:9) in the brand fonts and palette
- Document covers and report templates
- Social media templates for launch announcements

## Tokens for engineers

```bash
npm install @noon-business/tokens
```

Ships CSS custom properties, a Tailwind preset and the raw JSON. Setup guide:
[documentation](/resources/documentation).
