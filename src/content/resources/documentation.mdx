## Packages

| Package | Contains |
| --- | --- |
| `@noon-business/tokens` | CSS variables, Tailwind preset, JSON |
| `@noon-business/icons` | Stroke icons as React components |

## Setup

```bash
npm install @noon-business/tokens
```

```css
@import "tailwindcss";
@import "@noon-business/tokens/css";
```

Light mode only for now. Tokens are a single palette in `globals.css`.
See [color](/foundations/color).
