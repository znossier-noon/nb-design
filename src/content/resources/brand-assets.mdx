Everything here is export-ready and pre-approved. If you need an asset that
isn't here, ask the brand team, don't recreate it.

## Logo package

The package contains every approved mark in every approved variant:

- **Lettermark**, SVG and PNG (16–1024px), full color, all-white, all-ink
- **Wordmark**, SVG and PNG, light-background variants
- **Lockups**, lettermark + wordmark, horizontal and stacked
- **Platform icons**, iOS, Android and favicon sets with per-platform padding

Usage rules live on the [logo page](/foundations/logo), clear space, minimum
sizes and the misuse gallery.

## Icon library

The full stroke icon set ships as:

- Figma component set (in NB Foundations)
- SVG package and React components in `@noon-business/icons`

Style and usage rules: [iconography](/foundations/iconography).

## Imagery

- Photography direction: real merchants, real operations, regionally grounded
- Approved photography lives in the brand DAM (internal access)
- Illustration style: brand-soft palette, geometric, used for empty states
  and onboarding only
